## Prerequisites

  - Python 3
  - Nginx
  - uWSGI
  - Beanstalkd
  - Supervisord
  - MongoDB
  - A Pusher account
  - A Bugsnag account
  - A Gauges account
  - An ACRCloud account

## Installation

### Codebase

  1. Clone this repo
  2. Create a Python 3 virtualenv in the `venv` directory (at the root of the cloned repo)
  3. Then the usual `source venv/bin/activate` and `pip install -r requirements.txt`
