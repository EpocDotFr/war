Do I need to install some crap to make it working?

No. The only requirement is a modern web browser.

If you're still on Internet Explorer 7 or similar and you're experiencing issues, please either use a real web browser or go back to your middle of nowhere :)

How to capture the output from my speakers?

You have to create a fake recording peripheral (usually called "stereo mix") that will be accessed by WAR as if it were a microphone.

It is open-source?

Nope. I don't think I will open-source it because it doesn't provide any technically meaningful matter to the community.

Is WAR working on mobile devices?

Yes. Again, any modern mobile web browser will be OK.

Why would I use WAR instead of other services?

The WAR final goal isn't different from other services, but the way it's achieved is. The About page has more informations about this.
