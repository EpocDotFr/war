Wut?

Web Audio Recognizer (WAR) is a simple web page that allow you to recognize unknown music tracks and get the associated artist and title. Think about a Chazam, but web-based.

How?

You can either submit an audio file or capture / sing the music via your microphone, or even capture the music currently playing from your speakers (a little bit of configuration is needed for this, see bellow).

Do I need to install some crap?

Nothing to install. The only requirement is a modern web browser.

Why?

I created WAR primarly for myself, because I was frustated that Digitally Imported - a great web radio I listen for years - often play tracks that are mixes. Since mixes are several tracks mixed together in one huge track, you can't get the real tracks name.

As I use several methods at several places to listen to DI, I wanted a system that didn't need to install anything (other than a modern web browser).

I also needed that this system searches several audio databases in one time to maximize the chances to get positive matches.

How to capture audio from my speakers?

You have to create a fake recording peripheral (usually called stereo mix) that will be accessed by WAR like if it's a microphone.

It is open-source?

Nope. Not yet.
