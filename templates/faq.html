Do I need to install some crap to make it working?

No. The only requirement is a modern web browser. See below for a list of supported web browsers.

What are the supported web browsers?

Desktop

  • Firefox (version 17 or above)
  • Chrome (version 21 or above)
  • Opera (version 18 or above)
  • Vivaldi
  • Microsoft Edge

Mobile

  • Android Browser (Android 5 or above)
  • Chrome for Android
  • Firefox for Android
  • Opera Mobile
  • Blackberry Browser (version 10 or above)

How to capture the output from my speakers?

You have to create a fake recording peripheral (usually called "stereo mix") that captures all sounds from your computer to it. This way, it will be accessed by WAR seamlessly as if it were a microphone.

Here's a list of tutorials for each operating system to enable such a fake peripheral:

  • Windows Vista or above
  • Windows XP
  • Mac OS
  • Ubuntu

It is open-source?

Nope. Maybe in the future. But we use a huge number of open-source software and libraries (see on the About page).

Why would I use WAR instead of other services?

The final goal of WAR isn't different from other services (getting the track name of an audio sample), but the way it's achieved is. The About page has more information about how WAR works and why it has been created.

What about my privacy and the data sent to WAR?

See on the Terms of service page. Don't worry, there is no incomprehensible bullshit written on it.
