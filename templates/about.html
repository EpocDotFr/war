Web Audio Recognizer (WAR) is a simple web page that allows you to recognize unknown music tracks and get the associated artist and title. Think about Shazam, but web-based.

You can either capture or sing the music via your microphone. You can even capture the music currently playing from your speakers for more accuracy (a little bit of configuration is needed for this, please refer to the FAQ).

Author and genesis

I (Epoc) primarly created WAR for myself, because I was frustated that Digitally Imported - a great web radio I'm listening for years - often play tracks that are mixes. Since mixes are several tracks mixed together in one huge track, you can't get the real tracks titles.

As I use several methods at several places to listen to Digitally Imported, I wanted a system that didn't need to install anything.

I also needed this system to search several audio databases at once to maximize the chances of getting positive matches.

WAR is a side project started in June 2016. I plan to add new features and maintain it on long term. You can read about the planned features and their progression on the roadmap page.

Technologies

WAR is rolling on Python, Nginx and Ubuntu Server at Scaleway. It also uses Beanstalk, Supervisor, Pusher, MariaDB and Zepto.

It requests several audio databases for each recognition request, you can see the list on the stats page.
